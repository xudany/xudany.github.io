### 万分懊恼的 commit 信息手误拯救办法

commit信息写错，但是已经手比脑快地按了回车这种事情就是日常o(╥﹏╥)o

只要还没有push就还有换回的办法！！

走你┏ (゜ω゜)=☞：

```
git commit --amend
```

有了`--amend`妈妈再也不用担心我commit信息写错啦！赞(o゜▽゜)o☆

ps：以上命令执行后还需要按`i`键才可以进入insert模式，才可以输入，修改完成后按`Esc`键退出修改模式，然后按`:` + `wq！`强制退出并保存。
